# AlgoScape — Interactive Graph Algorithm Visualizer & Performance Analyzer

## Project Overview
**AlgoScape** is a desktop application that allows you to **visualize and compare classical graph algorithms** interactively.  
It demonstrates algorithm execution, graph traversal, shortest path finding, and minimum spanning tree construction, along with runtime benchmarking.

This project is built using **Python**, **CustomTkinter**, **NetworkX**, and **Matplotlib**, making it both **educational and visually appealing**.

---

## Features

- **Graph Generation**
  - Create random graphs of any size.
  - Adjust graph density to test algorithms on sparse or dense graphs.

- **Algorithms Included**
  - **Traversal:** BFS, DFS  
  - **Shortest Path:** Dijkstra, Bellman-Ford  
  - **Minimum Spanning Tree (MST):** Prim, Kruskal  

- **Visualization**
  - Interactive plotting of graphs.
  - Highlight current node in traversal or shortest path.
  - Visualize MST edges in red and shortest path in green.

- **Performance Analysis**
  - Benchmark runtime of algorithms on generated graphs.
  - Compare algorithm performance visually using bar charts.

- **Interactive GUI**
  - Built using **CustomTkinter** for a modern, responsive interface.
  - Choose algorithms, graph size, and density with sliders and dropdowns.
  - Draw your own graph manually and see results instantly.

---

## Tech Stack

- **Python 3.x**  
- **CustomTkinter** → GUI  
- **NetworkX** → Graph creation and algorithms  
- **Matplotlib** → Graph plotting and performance charts
